	\section{Surface water flow estimation}
Simulate surface water flow pattern using the derived DTM using AI.
    

Key objectives:
	\begin{itemize}
		\item generate a simplie analouge for water flow using iterative flow accumulation logic.
        \item use the flow pattern to identify natural flow paths and areas prone to waterlogging or flooding.
		\item Train a model to predict surface water flow patterns based on the derived DTM and other relevant features like rainfall data, soil type, and land use.
	\end{itemize}
